# UniEventDemoWebPage
A simple webpage I'm making to represent my department at a university event.

This project is written in HTML and CSS (mostly bootstrap) with javascript for the logical components.
It is coded by hand, however the javascript portion is AI generated, so sorry for the horrendous code.
I am very against AI usually, but due to my limited skills in programming and only really knowing Java, I was hit with a brick wall when writing this webpage.
So I did a weird thing, I wrote my logic in Java and then had the Jetbrains AI (integrated within the InelliJ idea IDE) sort of "translate it" into javascript for me. 
That way the code turned out to be somewhat readable, because the AI had a reference of sorts to build up on. And I got to build the website within the deadline.
